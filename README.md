## Overview

AI-Powered Software Project Risk Management Platform is a full-stack web application designed to help organizations identify, analyze, and mitigate software project risks. The platform combines project analytics, resource monitoring, budget tracking, quality assessment, and AI-driven recommendations to improve project success rates.

## Problem Statement

Software projects frequently face challenges such as budget overruns, schedule delays, resource overallocation, quality issues, and technical debt. Traditional project management systems primarily focus on task tracking and often lack intelligent risk assessment capabilities.

This platform addresses these challenges by providing real-time project insights, risk analysis, and AI-generated mitigation recommendations.

## AI Module

The platform integrates Ollama with the Llama 3.1 model to provide intelligent project risk mitigation recommendations.

The AI engine analyzes:

* Budget Usage
* Resource Utilization
* Defect Trends
* Project Progress
* Quality Metrics

Based on the analysis, the system generates actionable recommendations for project managers.
